# Exercícios de lógica de programação em Javascript

Criar uma classe 'calculosMatematicos' com 4 métodos: 
- calcularMedia
- somarDoisNumeros
- subtrairDoisNumeros
- multiplicarDoisNumeros

Criar outra classe para usar os 4 métodos criados:
- usarCalculosMatemáticos

Executar: 
(2 caminhos - pelo terminal ou pelo package.json criando um script)

# EXERCÍCIO 01

**ENTRADA:**
- Salário Bruto
- Tempo de casa em Meses

**PROCESSAMENTO:**
- Salário Bruto - (salario Bruto * 0.3) quando Tempo de Casa em Meses for menor que 36
- Salário Bruto - (Salario Bruto * 0.25) quando Tempo de Casa for maior ou igual que 36

**SAÍDA:**
- Salário Líquido

# EXERCÍCIO 02

**ENTRADAS:** 
- Salário
- Senioridade

**PROCESSAMENTO:**
- Quando a Senioridade for Sênior o Salário com Bônus será Salário + 500
- Quando Senioridade não for Sênior, o Salário com Bônus será apenas o Salário

**SAÍDA:**
- Salário com Bônus
