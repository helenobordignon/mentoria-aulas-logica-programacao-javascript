# Conceitos Fundamentais

**1. Interação Humano x Computador**

O computador não entende linguagem natural (como português ou inglês).
Ele precisa de instruções precisas, escritas em uma linguagem que consiga interpretar (ex.: JavaScript).
Você, como programador, é o tradutor entre o problema humano e a solução computacional.

**2. Instruções**

São comandos simples que dizem ao computador o que fazer.
Exemplo humano: “Pegue um copo, coloque água, beba.”
Exemplo em programação: console.log("Olá, mundo!"); → instrução para mostrar uma mensagem na tela.

**3. Decomposição**

É o processo de quebrar um problema grande em partes menores.

Exemplo: *Calcular salário líquido.*

Passo 1: Receber salário bruto.
Passo 2: Calcular descontos (INSS, IR).
Passo 3: Subtrair descontos.
Passo 4: Mostrar resultado.

Decompor ao máximo possível, e se atentar para as ações que são humanas e quais são do sistema.

ENTRADA > PROCESSAMENTO > SAÍDA

Decompor = quais ações humanas e quais são ações do sistema.
ações anda mais é do que "ALGORITMO", descrição passo a passo até o resultado.

**4. Algoritmos**
Um conjunto ordenado de instruções para resolver um problema.

Exemplo simples: Fazer café.
Esquentar água.
Colocar pó no filtro.
Jogar água quente.
Servir.

Em programação, o algoritmo é transformado em código.

💻 Conexão com **JavaScript**

Vamos usar esses conceitos para:
Criar *variáveis* (armazenar valores como salário, horas trabalhadas).
Fazer *operações* matemáticas (média, descontos, somas).
Usar *condições* (if/else) para verificar regras (ex.: se salário > limite, aplicar imposto).
Usar *loops* para repetir cálculos ou percorrer *listas*.
